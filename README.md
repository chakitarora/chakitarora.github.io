# chakitarora.github.io

Personal academic webpage of **Chakit Arora, PhD** — Post-Doctoral Research Fellow in Computational Biology at [Scuola Normale Superiore](https://www.sns.it), Pisa, Italy.

→ **[chakitarora.github.io](https://chakitarora.github.io)**

---

## What's here

A single-page academic site covering research profile, publications, software, and CV. Built as a static `index.html` — no framework, no build step, no dependencies to maintain.

Publications and citation metrics update automatically on every page load via the [OpenAlex](https://openalex.org) public API, keyed to ORCID [0000-0001-8613-0647](https://orcid.org/0000-0001-8613-0647). No manual updates needed when a new paper is indexed.

If the API is unreachable, a curated static list renders instantly as fallback.

---

## Stack

- Vanilla HTML / CSS / JS — no framework
- [Cormorant Garamond](https://fonts.google.com/specimen/Cormorant+Garamond) · [Inter](https://fonts.google.com/specimen/Inter) · [JetBrains Mono](https://fonts.google.com/specimen/JetBrains+Mono) via Google Fonts
- Live data: [OpenAlex REST API](https://docs.openalex.org) (free, no key required)

---

## To update

**New paper:** nothing. OpenAlex picks it up automatically within a few days of PubMed indexing.

**Profile details** (position, links, about text, software cards): edit `index.html` directly — all content is clearly commented.

**Photo:** replace `Chakit.jpg` in the root. The `<img>` tag falls back to initials if the file is missing.

## Design

The visual language draws on four threads held deliberately lightly: the mathematics of physical cosmology, cancer cell biology, Vedantic philosophy of consciousness and interconnection, and the sparse instrument-readout aesthetic of hard science fiction. The background lattice is Indra's Net — an ancient metaphor for infinite mutual reflection that turns out to describe protein interaction networks rather well. There's a longer explanation on the page itself.
